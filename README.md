# NIDS-Project
Machine Learning and Deep Learning-based Network Intrusion Detection System

## Requirements
- Python 3.10
- TensorFlow 2.x
- scikit-learn
- pandas, numpy, matplotlib, seaborn

## How to run
1. Clone repo: `git clone https://github.com/Bijoy-paul-01/NIDS-Project.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run `main.py` to train models and test detection.

## Dataset
NSL-KDD dataset included.
